#### はじめに
C#におけるメソッド（関数）は、プログラミングの基本中の基本です。メソッドは、特定のタスクを実行するコードのブロックを指します。この記事では、メソッドの基本概念、定義方法、使用方法について初心者向けに解説します。

#### メソッドとは？
メソッドは、特定の作業を行うためのコードの集まりです。メソッドを使用することで、コードの再利用性を高め、プログラムの構造を整理し、読みやすくすることができます。

#### メソッドの定義
C#でメソッドを定義する際には、アクセス修飾子、戻り値の型、メソッド名、そしてパラメーターを指定します。以下はシンプルなメソッドの例です。

```csharp
public int Add(int number1, int number2)
{
    return number1 + number2;
}
```
この例では、`Add`という名前のメソッドを定義しています。このメソッドは、2つの整数を受け取り、それらを加算した結果を返します。

#### メソッドの呼び出し
メソッドを定義したら、必要に応じてそれを「呼び出す」ことができます。以下は上記の`Add`メソッドを呼び出す例です。

```csharp
int result = Add(5, 3);
Console.WriteLine(result);  // 出力：8
```

#### パラメーター
メソッドにはパラメーターを指定できます。これにより、メソッドに値を渡し、その値に基づいて処理を行うことができます。パラメーターは必要に応じて任意の数だけ定義できます。

#### 戻り値
メソッドは、戻り値を返すことができます。これはメソッドが行った処理の結果を呼び出し元に渡すためのものです。戻り値がない場合は、`void`を指定します。

#### メソッドのオーバーロード
同じ名前のメソッドを異なるパラメーターで複数定義することを「オーバーロード」といいます。これにより、異なるタイプの引数に対応することができます。
